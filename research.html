Generating Molecules with Reinforcement Learning

Reinforcement Learning is a fundamental paradigm in machine learning that enables machine learning models to learn optimal decision-making strategies through interactions with an environment. We apply this framework to the chemical language space to generate novel small molecules that bind to the alpha-synuclein aggregation-prone interface, disrupting this proteinopathy.
